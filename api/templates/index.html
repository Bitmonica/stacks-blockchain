To start making calls to the Onename API, you'll need API credentials (an App ID and an App secret key).

Sign Up for an API Account

Users

summary

Returns the profile data (passcard) and verifications of the user with a given +passname.

http request

GET /v1/users/{passname}

response fields

profile, verifications

example request

TRY IT OUT

Libraries

Libraries are available in several languages.

curl http://api.onename.com/v1/users/fredwilson -u 'YOUR-API-ID:YOUR-API-SECRET'

Register

summary

Takes in a passname to be registered along with the address that will own the passname. Optionally, takes in the passcard data that should be associated with the passname being registered. Returns a status object that shows if the request was successfully received. It takes on the order of hours to actually complete the registration.

http request

POST /v1/users/{passname}/register

query parameters

transfer_address, passcard (optional)

response fields

status

example request

Libraries

Libraries are available in several languages.

curl http://api.onename.com/v1/users/fredwilson/register \
-u 'YOUR-API-ID:YOUR-API-SECRET' \
-d '{"transfer_address": "N8PcBQn...", "passcard": {"bio": "I am a VC"}}' \
-X POST
