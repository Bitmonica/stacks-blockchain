Lookup users

Looks up the data for one or more users by their usernames. In order to perform more than one lookup at once, include a set of comma-separated usernames in the URL in place of the single username.

query parameters

-

returns

Returns an object with a top-level key for each username looked up. Each top-level key contains an sub-object that has a "profile" field and a "verifications" field.

TRY IT OUT

Definition
GET https://api.onename.com/v1/users/{usernames}
Example Request
curl https://api.onename.com/v1/users/fredwilson
    -u 'YOUR-API-ID:YOUR-API-SECRET'

Register a user

Takes in a passname to be registered along with the address that will own the passname. Optionally, takes in the passcard data that should be associated with the passname being registered. Returns a status object that shows if the request was successfully received. It takes on the order of hours to actually complete the registration.

query parameters
transfer_address
The namecoin address that the passcard will be transferred to once it has been registered.
passcard
The JSON data that will be associated with the passcard during registration.
returns

Returns an object with a status.

Definition
POST https://api.onename.com/v1/users/{passname}/register
Example Request
curl http://api.onename.com/v1/users/fredwilson/register \
    -u 'YOUR-API-ID:YOUR-API-SECRET' \
    -d '{"transfer_address": "N8PcBQn...", "passcard": {"bio": "I am a VC"}}' \
    -H 'Content-type: application/json' \
    -X POST
Example Response
{
  "status": "success"
}
